# ClassroomGradeHelper

Um script em Python para calcular e analisar as notas dos alunos em várias matérias, fornecendo feedback sobre aprovação ou a necessidade de estudos adicionais.

## Recursos
- Inserir notas para várias matérias.
- Calcular a média das notas para cada matéria.
- Determinar se um aluno está aprovado ou se precisa de estudos adicionais (recuperação).

## Como Usar
1. Execute o script.
2. Insira o número de matérias.
3. Insira a média escolar necessária para aprovação.
4. Insira as notas para cada matéria quando solicitado.
5. Visualize os resultados, incluindo a média das notas e o status para cada matéria.

## Exemplo
```python
# Exemplo de entrada e saída
Quantidade de matérias: 3
Média escolar: 7.0
Nome da 1ª matéria: Matemática
Notas das provas de Matemática (separadas por espaço): 5 6 7
Nome da 2ª matéria: Ciências
Notas das provas de Ciências (separadas por espaço): 8 9 7
Nome da 3ª matéria: História
Notas das provas de História (separadas por espaço): 6 7 8

# Saída
Matéria: Matemática, Média: 6.00, Status: Recuperação
Matéria: Ciências, Média: 8.00, Status: Aprovado
Matéria: História, Média: 7.00, Status: Aprovado
